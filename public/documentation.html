Movie app myFlix REST API documentation summary

Request Endpoint URL HTTP method Query parameter in URL Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Return data about a single movie by title to the user /movies/title GET None A JSON object holding data for a single movie. A JSON object about description, genre, director, image URL, whether it's featured or not
Return data about a genre (description) by name/title (e.g., “Thriller") movies/genre/genreName GET None A JSON object holding data for a genre
Return data about a director (bio, birth year, death year) by name movies/director/directorName GET None A JSON object holding data for a director
Allow new users to register /users POST Definiton User information such as Name, Last Name, Age, Email, Password
Allow users to update their user info (username) /user/id PUT Definition User can change profile fields, focus on name
Allow users to add a movie to their list of favorites /user/id/movieTitle POST None Showing only a text that a movie has been added—more on this later
Allow users to remove a movie from their list of favorites /user/id/movieTitle DELETE None Showing only a text that a movie has been added—more on this later
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /user/id DELETE None Showing only a text that a movie has been added—more on this later
